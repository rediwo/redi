What is HTML Templates?

HTML Templates in Redi Server provide a powerful way to create dynamic web pages with:

  • Go template syntax for dynamic content
  • Server-side JavaScript execution with <script @server>
  • Layout system for consistent page structure
  • Built-in Vimesh Style for utility-first CSS
  • No build step required - just write HTML
